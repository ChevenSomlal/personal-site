# Cheven Somlal - Personal Portfolio

🌸 A modern, professional portfolio website with anime-inspired design elements.

**Live Site:** https://chevensomlal.github.io/personal-site/

## Features

### Design
- 🎨 Light natural color palette (sakura, bamboo, ocean tones)
- 🌊 Glassmorphism effects
- ✨ Smooth animations and transitions
- 🎌 Subtle anime-inspired elements
- 📱 Fully responsive design

### Advanced CSS Techniques
- CSS Custom Properties (Variables)
- Glassmorphism with backdrop-filter
- CSS Grid & Flexbox layouts
- Keyframe animations
- Gradient text effects
- Hover transformations
- Parallax scrolling effects

### Interactive Features
- Smooth scroll navigation
- Animated floating shapes background
- Rotating avatar rings
- Timeline with hover effects
- Skill tags with hover states
- Project card tilt effect (3D)
- Cursor trail effect
- Stats counter animation
- Mobile hamburger menu
- Scroll-based navbar effects

### Accessibility
- Reduced motion support
- Semantic HTML
- ARIA labels
- Keyboard navigation friendly

## Tech Stack

- HTML5
- CSS3 (Advanced)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter, Fira Code)
- GitHub Pages (Hosting)

## Sections

1. **Hero** - Introduction with animated avatar
2. **About** - Bio with code window display
3. **Experience** - Timeline of career history
4. **Skills** - Categorized technology stack
5. **Projects** - Featured work including enterprise projects
6. **Contact** - Links to email, LinkedIn, GitHub

## Local Development

```bash
# Clone the repository
git clone https://github.com/ChevenSomlal/personal-site.git
cd personal-site

# Open in browser
open index.html
```

## Deployment

Site is automatically deployed to GitHub Pages on push to `main` branch.

## Customization

### Add Your Avatar
Replace the avatar placeholder in `index.html`:
```html
<div class="avatar-placeholder">
    <img src="your-photo.jpg" alt="Cheven Somlal">
</div>
```

### Add More Projects
Duplicate project cards in the Projects section and update content.

### Customize Colors
Edit CSS variables in `:root` of `styles.css`.
